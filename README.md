📜 **Retro Forum - Community Discussion Platform**

Retro Forum is a sleek, dynamic discussion platform that blends a classic forum aesthetic with a high-performance modern frontend. Built using Tailwind CSS and JavaScript, it provides a seamless user experience for browsing topics, engaging in discussions, and managing a personalized reading list.

🌟 Key Features
Dynamic Post Feed: Fetches and displays forum posts dynamically, allowing users to browse various categories like Comedy, Coding, and Lifestyle.

Real-time Search: An interactive search bar that filters through posts instantly using JavaScript to find specific topics or tags.

Mark as Read / Queue: A unique feature where users can click an icon to add posts to a "Reading List" sidebar, tracking view counts and titles in real-time.

Responsive Tailwind UI: Fully mobile-responsive layout featuring custom-styled cards, badges, and a "Retro" themed color palette.

Loading States: Implements interactive loading indicators (spinners) to improve perceived performance during data fetching.

💻 Technology Stack
Structure: HTML5

Styling: Tailwind CSS (Utility-first styling)

Interactivity: Vanilla JavaScript (ES6+ Fetch API & DOM Manipulation)

Icons: FontAwesome
